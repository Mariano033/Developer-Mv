Aprende a Programar

Nivel 1: Fundamentos de HTML, CSS y JavaScript (6 clases)

HTML básico: Estructura de una página web, etiquetas principales, enlaces, imágenes.

CSS básico: Estilización de páginas, selectores, propiedades, layout básico (box model, flexbox).

JavaScript básico: Variables, tipos de datos, operadores, estructuras de control.

Nivel 2: Programación avanzada y sitios dinámicos con JS (8 clases)

JavaScript avanzado: Funciones, objetos, arreglos, manipulación avanzada del DOM.

Eventos y manipulación del DOM: Listeners, cambios dinámicos en el contenido web

Introducción a APIs: Consumo de datos de APIs, manejo de respuestas.

Nivel 3: Desarrollo de aplicaciones web completas (12 clases)

Introducción a React: Componentes, JSX, props y estados


Manejo de rutas: React Router para crear aplicaciones con múltiples vistas.


Buenas prácticas y despliegue: Optimización de código, Git, deploy en plataformas como Netlify o Vercel.
